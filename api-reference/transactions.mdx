<Info>Requires a **PRO API key**.</Info>

Generates ready-to-sign transaction calldata for any supported vault action. Pass the output directly to your wallet provider — no ABI encoding required.

Covered here: `/v2/transactions/context`, `/v2/transactions/{action}`, `/v2/transactions/rewards/context`, `/v2/transactions/rewards/claim`.
