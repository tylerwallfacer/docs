title: "Benchmarks"
description: "DeFi benchmark APY rates for USD and ETH strategies, including historical time-series."
---

<Info>Requires a **PRO API key**.</Info>

Reference rates representing baseline DeFi yield for USD and ETH strategies. Use these to compare vault performance against the broader market.

| Method | Endpoint | Description |
|--------|----------|-------------|
| `GET` | [/v2/benchmarks/\{network\}](/api-reference/benchmarks-pro/get-benchmark-rate) | Current benchmark rate for a network |
| `GET` | [/v2/historical-benchmarks/\{network\}](/api-reference/benchmarks-pro/get-historical-benchmarks) | Historical benchmark rates for a network |
