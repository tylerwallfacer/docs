---
title: "Detailed Vaults"
description: "Comprehensive vault data including APY, TVL, risk scores, fees, holder data, and protocol details."
---

Full vault objects with performance metrics, reputation scores, fee structures, holder data, and protocol info. Use these endpoints to build vault comparison views, detail pages, or yield aggregators. No PRO key required.

Covered here: `/v2/detailed-vaults`, `/v2/detailed-vaults/{network}/{vaultAddress}`, and the `/apy` and `/tvl` sub-endpoints.
